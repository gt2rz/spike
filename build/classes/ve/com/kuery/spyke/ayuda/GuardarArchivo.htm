GUARDAR ARCHIVO

 

Para guardar un archivo el usuario debera presionar sobre el boton guardar de la barra de herramientas o sobre el item guardar del menu archivo.

 

 

 

Y aparecera la siguiente ventana.

 
Luego debera asignarle un nombre al archivo que desee guardar y pulsar sobre el boton guardar para continuar con el proceso de guardado o el boton cancelar para detener el proceso de guardado.
